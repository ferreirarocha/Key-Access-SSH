## Instalação automática
```
curl -s https://raw.githubusercontent.com/ferreirarocha/Key-Access-SSH/master/kas | bash -s -- --install
```

## Instalação manual
```
wget https://raw.githubusercontent.com/ferreirarocha/utilitarios/master/kas
```

```
chmod +x kas
```

```
cp /usr/bin/
```
